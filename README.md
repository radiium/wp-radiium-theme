# wp-radiium-theme

Simple wordpress theme for my personal website [radiium.space](http://radiium.space)

## Development:
This theme use sass (source in [assets/css](./assets/css))


#### Install deps
```console
npm install
```

#### CSS:
```console
# Compil once
npm run css
# Watch change
npm run watch
```

#### JS:
```console
# Minify
gulp compress
```
