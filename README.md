# wp-radiium-theme

Simple wordpress theme for my personal website [radiium.space](http://radiium.space)

## Development:
This theme use sass (source in [assets/css](./assets/css))


#### Install deps
```console
npm install
```

#### CSS:
```console
# Compil SASS files
gulp sass
# Watch SASS files change
gulp sass:watch
```

#### JS:
```console
# Minify
gulp compress
```
